Ejercicio 1

listado de todos los usuarios registrados en la base de datos.

  1. Nombres.
  2. Apellidos.
  3. Correos.
SELECT names, lastname, email FROM users;

Ejercicio 2

Listado de departamentos registrados en la base de datos.

  1. Nombre del departamento.
  2. Nombre del país.
SELECT departments.name AS department_name, countries.name AS country_name FROM departments INNER JOIN countries ON departments.countries_id = countries.id;

Ejercicio 3

listado de escuelas por un pais determinado.

  1. Nombre de la escuela.
  2. Nombre del país.
SELECT schools.name AS school_name, countries.name AS country_name FROM schools INNER JOIN countries ON schools.countries_id = countries.id WHERE countries.id = 2

Ejercicio 4

listado de usuarios ordenados de mayor edad a menor edad.

  1. Nombres.
  2. Apellidos.
  3. Fecha de nacimiento.
SELECT names, lastname, date_birth FROM users ORDER BY date_birth ASC;

Ejercicio 5

en que departamentos hay más usuarios registrados.

  1. departamento.
  2. Id.
SELECT departments_id, COUNT(*) AS total_users FROM users GROUP BY departments_id ORDER BY total_users DESC LIMIT 1;

Ejercicio 6

cual es el país con mas usuarios registrados.

  1. País.
  2. Id.
SELECT c.name AS countries_name, COUNT(u.id) AS user_count FROM users u JOIN countries c ON u.departments_id = c.id GROUP BY c.name ORDER BY COUNT(u.id) DESC LIMIT 1;

Ejercicio 7

listar solo 3 departamentos.

  1. Nombre del departamento
  2. Id.
SELECT name, id FROM departments LIMIT 3;

Ejercicio 8

listar la informacion de un usuario determinando.

  1. Nombres
  2. Apellidos
  3. Correo
  4. Nombre del colegio
  5. Nombre del departamento
  6. Nombre del país
SELECT u.names, u.lastname, u.email, s.name AS school_name, d.name AS department_name, c.name AS country_name FROM users u JOIN schools s ON u.schools_id = s.id JOIN departments d ON u.departments_id = d.id JOIN countries c ON s.countries_id = c.id WHERE u.id = 3;

Ejercicio 9

listado de usuarios con un nombre determinado "and".

  1. Nombres
  2. Apellidos
SELECT names, lastname FROM users WHERE names = 'amparo';

Ejercicio 10

Cuantos usuariosviven en un departamento determinado.

  1. Cantidad de usuarios
  2. "números"
SELECT COUNT(*) AS cantidad_usuarios, departments.name AS nombre_departamento FROM users JOIN departments ON users.departments_id = departments.id WHERE departments.id = 5 GROUP BY departments.id;

Ejercicio 11

Listado de usuarios que tengan mas de 20 años.

  1. Nombres.
  2. Apellidos.
  3. Fecha de nacimiento.
SELECT names, lastname, date_birth FROM users WHERE DATE_ADD(date_birth, INTERVAL 20 YEAR) <= CURDATE();

Ejercicio 13

Listado de colegios que tengan el número 7 en su número de telefono.

SELECT * FROM schools WHERE phone LIKE '%7%';

Ejercicio 14

Listado de usuarios que no esten asociados a ningún departamento.

SELECT u.names, u.lastname FROM users u LEFT JOIN departments d ON u.departments_id = d.id WHERE d.id IS NULL;

Ejercicio 15

Listado de usuarios que no esten asociados a ningún colegio.

SELECT names, lastname FROM users WHERE schools_id IS NULL;
